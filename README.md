# Recrutation task
Script to parse and analyze data from https://blog.hubspot.com. <br/>

## Getting Started

1. Git clone repo
```
git clone https://github.com/saworz/hubspot-fetcher.git
```
2. Move to repo dir
```
cd hubspot-fetcher
```
3. Run from terminal
```
python -m pip install -r requirements.txt
python main.py
```
